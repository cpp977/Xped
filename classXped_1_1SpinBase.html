Detailed Description

template<typename Symmetry_, std::size_t order = 0>
class Xped::SpinBase< Symmetry_, order >

This class provides the local operators for general spins with quantum number S.

t: \(t\)
V: \(V\)
Vz: \(V_z\)
Vxy: \(V_{xy}\)
